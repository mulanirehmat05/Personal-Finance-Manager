# 💰 Personal Finance Manager

A desktop-based Personal Finance Manager built using Java Swing for tracking income and expense transactions with category filtering, sorting, and persistent local storage.

---

# 🚀 Features

- Add, Edit, and Delete Transactions
- Income & Expense Tracking
- Category-Based Filtering
- Date-Wise Transaction Sorting
- Dynamic Balance Calculation
- Local File Storage using File Handling

---

# 🛠 Tech Stack

- Java
- Java Swing
- OOP Concepts
- File Handling

---

# 📂 Project Structure

- UI Layer
- Model Layer
- Data Handling Layer

---

# ▶️ Run Project

1. Open the project in IntelliJ IDEA
2. Run the main application file
3. Start managing transactions
